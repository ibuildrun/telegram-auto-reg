# Telegram Auto-Regger - Быстрый старт

## Что нужно скачать

1. **TelegramAutoReg.exe** - основная программа
2. **config.yaml.example** - файл настроек

---

## Установка (5 минут)

### Шаг 1: Создай папку

Создай папку для программы, например:
```
C:\TelegramAutoReg\
```

Положи туда оба скачанных файла.

### Шаг 2: Переименуй файл настроек

Переименуй `config.yaml.example` в `config.yaml`

### Шаг 3: Заполни настройки

Открой `config.yaml` в Блокноте и заполни обязательные поля:

```yaml
# SMS провайдер - получи API ключ на https://sms-activate.org
sms:
  provider: "sms-activate"
  api_key: "ТВОЙ_API_КЛЮЧ"

# Telegram API - получи на https://my.telegram.org/apps
telethon:
  api_id: 12345678
  api_hash: "твой_api_hash"

# ADB - укажи адрес эмулятора
adb:
  device_udid: "127.0.0.1:5555"
```

### Шаг 4: Установи дополнительные программы

| Программа | Зачем нужна | Где скачать |
|-----------|-------------|-------------|
| LDPlayer | Android эмулятор | https://www.ldplayer.net/ |
| ExpressVPN | Ротация IP (опционально) | https://www.expressvpn.com/ |

### Шаг 5: Настрой эмулятор

1. Запусти LDPlayer
2. Зайди в Настройки Android -> О телефоне -> 7 раз нажми на "Номер сборки"
3. Вернись в Настройки -> Для разработчиков -> Включи "Отладка по USB"
4. Установи Telegram из Play Store

### Шаг 6: Запусти программу

Дважды кликни на `TelegramAutoReg.exe`

## Регистрация аккаунтов

1. Убедись что эмулятор запущен
2. Перейди в раздел **Registration**
3. Выбери страну для номера
4. Укажи максимальную цену за SMS
5. Нажми **Start Registration**

Программа автоматически:
- Получит номер телефона
- Введёт его в Telegram
- Получит SMS код
- Завершит регистрацию
- Сохранит сессию

## Решение проблем

### Программа не запускается
- Убедись что Windows 10/11
- Попробуй запустить от имени администратора

### ADB не видит эмулятор
- Проверь что эмулятор запущен
- В Config Check нажми "Restart ADB"
- Проверь device_udid в config.yaml

### SMS не приходит
- Попробуй другую страну
- Увеличь max_price в настройках
- Проверь баланс на sms-activate.org

### Где смотреть логи
- Файл `telegram_regger.log` в папке с программой

## Где получить API ключи

### SMS-Activate API
1. Зарегистрируйся на https://sms-activate.org
2. Пополни баланс
3. Перейди в Профиль -> API
4. Скопируй API ключ

### Telegram API
1. Зайди на https://my.telegram.org
2. Войди по номеру телефона
3. Перейди в "API development tools"
4. Создай приложение
5. Скопируй api_id и api_hash
